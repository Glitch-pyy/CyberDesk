# CyberDesk

CyberDesk is an open-source modular desktop hardware system based on ESP32.

## Goals

- Desktop status dashboard
- Custom Stream Deck
- Plugin-based hardware modules
- Web configuration panel
- OTA firmware update
- Future PCB and 3D-printed case
